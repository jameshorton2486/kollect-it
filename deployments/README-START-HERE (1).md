## 🎯 The Key Concept: Token System

### Old Way ❌ (Inconsistent)
```tsx
// Hardcoded everywhere, inconsistent
<header className="bg-[#FFFFFF]">
<h1 className="text-[#1E1E1E]">
<p className="text-[#B1874C]">
<button className="bg-[#1E3A5F]">
<div className="border-[#E0DDD9]">

// Problem: Change one color = find & replace everywhere
//          Easy to miss one
//          Hard to maintain
```

### New Way ✅ (Clean & Maintainable)
```tsx
// Semantic tokens, single source of truth
<header className="bg-white">
<h1 className="text-ink">
<p className="text-gold">
<button className="bg-cta">
<div className="border-neutral">

// Benefit: Change token in globals.css = updates everywhere
//          Single source of truth
//          Easy to maintain
//          Self-documenting
```

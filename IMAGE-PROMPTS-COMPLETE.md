## ✅ GENERATION SETTINGS

When using AI image generators:

**DALL-E / Midjourney Settings:**
- Aspect ratio: Square (1:1) for Instagram, Tall (2:3) for Pinterest
- Style: "Professional photography," "Studio lighting," "Fine art aesthetic"
- Quality: High detail, emphasis on texture and lighting
- Avoid: Generic, clip-art looking, overly saturated colors

**Style Consistency:**
- Use same aesthetic across all images (warm lighting, sophisticated styling)
- Keep color palette: Deep backgrounds, gold/bronze accents, warm wood
- Maintain: Museum-quality lighting, attention to detail

## 🚀 USING THESE PROMPTS

1. **Copy the entire prompt text** (the description, style notes, everything)
2. **Paste into image generator** (DALL-E, Midjourney, etc.)
3. **Adjust specifics** if needed:
   - Change "rare book" to specific item you have
   - Adjust colors if you prefer different mood
   - Add/remove details based on results
4. **Download high-resolution** (for both web and social)
5. **Save in organized folders:**
   ```
   /images
     /categories
     /homepage
     /social-instagram
     /social-pinterest
     /email
   ```

## 💡 PRO TIPS

- **Consistency matters:** Use same photographer/generator for category images
- **Lighting is 80%:** Better lighting > better styling
- **Details sell:** Show condition, patina, craftsmanship clearly
- **Multiple angles:** Generate variations (different angles, lighting) and pick best
- **Mobile-first:** Test all images on phone (how they look in feed)
- **Original content:** Your actual items > generic stock photos
- **Repurpose:** Use best photos across Instagram, Pinterest, email, site

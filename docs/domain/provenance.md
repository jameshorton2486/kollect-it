# Provenance Documentation

**Status:** Active  
**Last Updated:** December 2024

## Overview

Provenance (ownership history) is critical for high-value collectibles and antiques.

## Documentation Requirements

When available, document:
- Previous owners
- Acquisition dates
- Authentication certificates
- Appraisal documents
- Exhibition history

## Storage

- Provenance notes in `Product.productNotes`
- Documents stored in `Product.provenanceDocUrl`
- Appraisal URLs in `Product.appraisalUrls[]`

## Trust & Safety

- Provenance increases buyer confidence
- Verified provenance commands premium pricing
- Missing provenance is acceptable but must be noted

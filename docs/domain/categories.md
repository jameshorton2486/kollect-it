# Category Taxonomy

**Status:** Active  
**Last Updated:** December 2024

## Overview

Kollect-It uses a stable, hierarchical category taxonomy for product organization.

## Primary Categories

1. **Fine Art** (`fine-art`)
   - Paintings
   - Prints
   - Sculptures
   - Mixed media

2. **Rare Books** (`rare-books`)
   - First editions
   - Signed copies
   - Limited editions
   - Antiquarian books

3. **Collectibles** (`collectibles`)
   - Curated objects
   - Curiosities
   - Memorabilia
   - Vintage items

4. **Militaria** (`militaria`)
   - Historical military artifacts
   - Uniforms
   - Medals
   - Documents

## Category Codes (for SKU)

- `FA` - Fine Art
- `RB` - Rare Books
- `CL` - Collectibles
- `ML` - Militaria

## Rules

- Categories are **immutable** once products are assigned
- New categories require approval and migration plan
- Subcategories can be added without breaking changes

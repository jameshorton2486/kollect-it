🔧 PRICING ENGINE DETAILS
═══════════════════════════════════════════════════════════════════════════════

   THREE-SOURCE WEIGHTING:
   ┌──────────────────────────────────────────────────────────┐
   │  Final Price = AI(50%) + Historical(30%) + Market(20%)   │
   └──────────────────────────────────────────────────────────┘

   CONFIDENCE CALCULATION:
   ┌──────────────────────────────────────────────────────────┐
   │  Base Score (AI Confidence)                              │
   │    + Excellent AI Analysis        (+25 if ≥90%)         │
   │    + Comparable Sales Found       (+20 if ≥5 comps)     │
   │    + Price Range Agreement        (+10 if tight)        │
   │    - Rare Item Assessment         (-20 if very rare)    │
   │    ± Market Trending              (±10)                 │
   │  ────────────────────────────────────────────────────   │
   │  = FINAL CONFIDENCE SCORE (0-100)                       │
   └──────────────────────────────────────────────────────────┘

   PRICE MULTIPLIERS:
   • Category: 0.8x (Glass) → 2.5x (Fine Art)
   • Condition: 0.3x (POOR) → 1.0x (EXCELLENT)
   • Rarity: 0.8x (COMMON) → 4.0x (EXTREMELY_RARE)
   • Age: 0.9x (Recent) → 2.0x (Very Old)
   • Decade Bonus: +15% (1960s Art Deco, etc.)

   PRICE RANGES:
   • High Confidence (≥80%): ±15%
   • Medium Confidence (60-79%): ±25%
   • Low Confidence (<60%): ±35%


🌐 API ENDPOINTS
═══════════════════════════════════════════════════════════════════════════════

   GET  /api/admin/products/queue
        → Fetch pending products with pagination & filtering
        Params: page, limit, status

   POST /api/admin/products/approve
        → Approve product & create marketplace listing
        Body: { productId, finalPrice }

   POST /api/admin/products/reject
        → Reject product with optional reason
        Body: { productId, reason? }

   POST /api/admin/products/bulk-approve
        → Bulk approve multiple products
        Body: { productIds[], useSuggestedPrices, priceOverride? }

   GET  /api/admin/products/history
        → Fetch approval/rejection history
        Params: page, limit, status, startDate?, endDate?
